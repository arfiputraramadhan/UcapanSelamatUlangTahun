# Website Ucapan Selamat Ulang Tahun dengan Kejutan

Website interaktif untuk membuat ucapan ulang tahun yang personal dengan berbagai efek kejutan visual dan audio.

DEMO WEB NYA= https://privatserver.diskon.com

## Fitur


1. **Input Data Personal**:
   - Nama penerima
   - Tanggal, bulan, dan tahun lahir
   - Pesan tambahan khusus
   - Pilihan tema warna
   - Pilihan jenis kejutan

2. **Perhitungan Otomatis**:
   - Usia berdasarkan tanggal lahir
   - Hari menuju ulang tahun berikutnya
   - Zodiak berdasarkan tanggal lahir
   - Shio Tionghoa berdasarkan tahun lahir
   - Hari kelahiran (Senin, Selasa, dll.)

3. **Efek Kejutan**:
   - Konfeti animasi
   - Balon terbang
   - Kembang api digital
   - Musik latar "Happy Birthday"
   - Pesan rahasia dengan fakta menarik

4. **Kontrol Interaktif**:
   - Tombol untuk memicu efek kejutan
   - Kontrol musik (putar/hentikan)
   - Pilihan tema warna yang dapat diubah
   - Reset untuk membuat ucapan baru

5. **Responsif & Mobile-Friendly**:
   - Tampilan optimal di desktop, tablet, dan ponsel
   - Touch-friendly dengan target sentuh minimal 44px
   - Animasi yang dioptimalkan untuk performa mobile
   - Dukungan landscape dan portrait mode
   - Efek visual yang disesuaikan berdasarkan ukuran layar

## Cara Menggunakan

1. Buka file `index.html` di browser web
2. Isi data penerima di form:
   - Nama lengkap
   - Tanggal lahir (tanggal, bulan, tahun)
   - Pilih tema warna
   - Pilih jenis kejutan yang diinginkan
   - Tambahkan pesan pribadi (opsional)
3. Klik tombol "Buat Kejutan Ulang Tahun!"
4. Lihat kartu ucapan yang dibuat khusus
5. Gunakan tombol kontrol untuk memicu efek kejutan
6. Klik "Buat Ucapan Baru" untuk mereset dan membuat yang baru

## Penyimpanan Data

Website ini menggunakan localStorage browser untuk:
- Menyimpan data terakhir yang dimasukkan
- Memuat ulang data saat halaman dibuka kembali (dalam 7 hari)
- Data tetap bersifat lokal dan tidak dikirim ke server
